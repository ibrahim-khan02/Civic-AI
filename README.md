# 🚀 CivicAI

### Smart Complaint Management System

CivicAI is a web-based platform that allows citizens to submit complaints related to civic issues (like water, roads, electricity) and track their status efficiently. It also includes a simple chatbot to assist users with common queries.

---

## 📌 Features

* 📝 Submit complaints easily
* 📊 Dashboard with real-time stats

  * Total complaints
  * High priority
  * Pending
  * Resolved
* 🏷️ Automatic categorization & priority tagging
* ✅ Mark complaints as resolved
* 🤖 Chat assistant for basic queries
* 💾 Backend API integration
* 🎨 Clean UI using Bootstrap

---

## 🛠️ Tech Stack

*  **Frontend:** HTML, CSS, JavaScript
* **UI Framework:** Bootstrap
* **Backend:** Node.js, Express 
* **Database:** MongoDB
